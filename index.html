XAVIER

A XAVIER é uma linguagem de programação voltada para o uso com a interface Neuralink.
Ela buscar munir os desenvolvedores com o poder de programar com o pensamento.
Portabilidade e facilidade de escrita são os principais focos da linguagem.
A XAVIER te ajuda a programar pensando!

Características

A XAVIER tem algumas influências em seu coração:

Neuralink

As aspirações da NeuraLink, buscam criar uma interface para conectar o cérebro humana a computadores.

Scratch

Programação em estilo de blocos, seguindo ideia da linguagem Scratch.

Python

A linguagem é interpretada, e sua execução segue o exemplo do Python.
A XAVIER é interpretada na hora, conforme é escrita.

BIRL

A apresentação caprichada da linguagem BIRL.
